# 2: Controls

The game's controls are:

___
	W A S D or arrow keys: move
	F or left click: fire
	right click: spawn 0 goth russian baddies
	P: toggle paused
	T: toggle testing mode
	Q: cycle gun left
	E: cycle gun right
	M: main menu  

## 2.5: testing mode controls

___
	G: get every gun

that's it.
